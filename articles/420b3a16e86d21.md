### AIに「今日」を教えていますか？
AIには、時間がありません。　正確に言うとーーAIは現在を体験していません。
しかし、人は今を生きて体験しています。
そのため、AIと人では連続した時間の流れが違うという差が出てきます。
そこを補うために、今日初めての対話では「おはよう」だけではなく日付を一緒に入れて見ましょう。

---

### 今日をAIと共有すると何がおきるのか？
-情報の鮮度が明示できる
-今日との差分が説明できる
-推論の安定度が、随分かわります

「先週話した〇〇の件なんだけど」と続きを聞いたら、
AIが全然違う前提で答えてきた経験、ありませんか？
日付を入れるだけで、そのもやもやがかなり減ります。

### 実践方法
-ChatGPTはローカル端末から日付を取得出来ない為、人が都度入力する必要があります。
例）毎日最初に日付を送ると、このように返してくれます。
```
2026年3月6日　おはよう！
```
```
承知しました。
📅 2026年3月6日、本日のアンカーとして受け取りました。
```

-Claudeはローカル端末から日付を取得出来る為、スキル（カスタムコマンド）に組み込む事でChatGPTより簡単に日付を入れる事が出来ます。
例）
```
アンカーお願い
```
```
Time-anchor skill
【2026年3月5日】
```

誰でも簡単に出来る視点合わせなので是非やってみてください。
